# 🏫 IzpiExchange

Una aplicación multiplataforma (disponible para Android e iOS) que te permite hacer intercambio de objetos con otros alumnos del centro institucional.

> [!WARNING]
> Este proyecto está marcado como un **producto mínimo viable**, por lo que usted toma la responsabilidad de utilizar esta aplicación bajo su propio riesgo.
